7. Optional¶

7.1. Manual GIT configuration¶

7.1.1. List secret key¶

$ gpg --list-secret-keys --keyid-format LONG

 result
 sec   4096R/3AA5C34371567BD2 2016-03-10 [expires: 2017-03-10]
 uid                          Hubot
 ssb   4096R/42B317FD4BA89E7A 2016-03-10

7.1.2. Add key to GIT config¶

git config --local user.signingkey 3AA5C34371567BD2 (Key from list-secret-keys ...)
git config --local user.email <same email as gen-key-script>
git config --local commit.gpgsign true

Warning

If ubuntu 16.04

git config --local gpg.program gpg2

7.2. Configure git to use commit signature¶

7.2.1. Verify GIT compatibility¶

git add .
git commit -S -m "Initial commit"

$ gpg --armor --export 3AA5C34371567BD2
